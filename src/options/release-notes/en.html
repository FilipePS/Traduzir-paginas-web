1.4 - February 11, 2020

In this version, the Google Translate Widget is injected but is hidden, the objective is that you use the Widget options through the extension's interface.

In the extension interface there were only two options:

Translate
This option injected the Google Translate Widget into the page, this widget was hidden.
Show original
Restores the page, but the Widget is still injected.

There was nothing left, not even an option to change the target language.

1.0, 1.2, 1.3 - February 9, 2020

The first version was released on February 9, 2020.

Versions 1.2 and 1.3 also existed and were released on the same day as version 1.0, I don't know what exactly was changed in them but it was insignificant.

The only thing the extension did in this version was inject the Google Translate widget into the website.

This Google Widget allows you to translate the website into any language with the same quality as the Chrome translator.

I learned a lot about HTML, CSS and JAVASCRIPT on the W3Schools website. And it was even thanks to W3Schools that I learned that this Google Translate Widget existed and how to use it, perhaps without this site, TWP would never have existed.
